Module: json-replace-exponentials

(require("json-replace-exponentials"))(json, replaceropt) → {string}

Replaces numbers in exponential notation in a given JSON string.
Parameters:
Name Type Attributes Description
json string JSON in which to replace numbers.
replacer function <optional>
 Optional replacer function called with a number in exponential format returning a string which will replace the number in the return value.

Throws:
  • If json is not a string.
    Type
    TypeError
  • If replacer is not a function.
    Type
    TypeError
  • If replacer is undefined and a number in exponential format has an exponent which is larger than 1,000 or smaller than -1,000. (To mitigate risks from unexpected large size increase in output.)
    Type
    RangeError
Returns:
Input JSON with numbers in exponential format replaced by fixed-point format, or by replacer, if provided.
